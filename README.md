# キャリアサマリ

リクルート（旧 住まいカンパニー）での不動産営業を起点に、飲食業界の人事、SaaSスタートアップでのプロダクトマネージャー（PdM）および事業開発を経て、現在はEngineer HRに従事しています。ビジネスサイド、プロダクトサイド、コーポレートサイドの多角的な視点を持ち、組織と事業の成長を繋ぐ役割を専門としています。

## 職務経歴の概要

- **不動産営業（株式会社リクルート住まいカンパニー）**
  - 不動産領域における提案営業に従事し、顧客志向の課題解決と目標達成に向けたプロセス管理の基礎を習得。

- **人事（飲食業界）**
  - 採用から制度設計、組織開発まで幅広く担当。現場の課題に寄り添った人事施策を推進。

- **PdM・事業開発（SaaSスタートアップ）**
  - プロダクトマネージャーとしてロードマップの策定やUX改善を主導。併せて事業開発として新規市場の開拓やアライアンス構築、ビジネスモデルの検証に従事。

- **Engineer HR（現在）**
  - エンジニア採用戦略の立案・実行、技術広報、エンジニア組織のエンゲージメント向上など、開発組織の競争力強化をミッションに活動。
