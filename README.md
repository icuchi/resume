# 職務経歴書：樋口 愛（Ai Higuchi）

### **People × Organization × Product Architect**

**「事業の成長」と「組織の構造」を接続し、プロダクトの価値を最大化する仕組みを創る**

## 1｜キャリアサマリー

📍 **三位一体の構造設計力**
ビジネス（事業開発）、プロダクト（Pm/PdM）、コーポレート（人事/組織開発）の3領域を横断。単なるオペレーション実行ではなく、各領域の結節点における「歪み」を解消し、再現性のある仕組み（構造）を構築することを得意とする。

📍 **エンジニア組織の成長支援（Engineer HR / DevRel）**
エンジニアの専門性とプロダクト特性を深く理解した上での採用要件定義、選考プロセスの設計の基盤構築に従事。事業サイドと開発サイドの「言語」を翻訳し、組織としての突破力を高める。

📍 **「型化」による組織改善の実績**
リクルートでの営業手法の標準化、飲食企業での人事DX推進、SaaSスタートアップでのBPO体制構築など、属人性を排除し、持続可能な運用フローへ落とし込む設計力に定評がある。



## 2｜職務経歴

### 【現在】パラレルワーク / 専門領域の構造設計 (2023.11 - Present)

現在は「人 × 組織 × プロダクト」の交差点において、以下の2軸で設計・アドバイザリー業務を推進中。

#### ① ECスタートアップ：エンジニア採用・組織設計アドバイザー

* **役割：** 採用戦略の構造化およびプロセス設計
* **実績/取り組み：**
* **エンジニア採用の要件定義と連携強化**： 採用要件の言語化を行い、現場エンジニアと人事間の連携フローを再設計。ミスマッチの低減と選考スピードの向上を実現。
* **採用広報ツールの最適化**： カンパニーデック（採用候補者向け資料）等の採用ツールをアップデート。情報の透明性を高め、カジュアル面談からの選考意向率を改善。
* **テクノロジーによるオペレーション効率化**： AI（ChatGPT等）やZapierを活用し、スカウト文面の作成や候補者管理の自動化フローを構築。ノンコア業務を削減し、候補者との対話時間を最大化。


#### ② HR SaaS（株式会社スタメン等）：事業開発・オペレーション設計

* **役割：** 新規事業責任者 / 事業開発部 部長（福利厚生サービス・BPO領域）
* **実績/取り組み：**
* **福利厚生事業「TUNAGベネフィット」の0→1立ち上げとPdM実務**：
事業責任者として、戦略策定（KPI管理・マーケ・営業）からオペレーション設計までを垂直立ち上げ。PdMとして要件定義や仕様策定を主導し、エンジニア一人ひとりの特性に合わせたコミュニケーションを徹底することで、開発プロジェクトの円滑な推進とリリースを実現。
* **BPO事業の新規立ち上げと収益化の牽引**：
事業開発部 部長として、福利厚生領域に加え、バックオフィスBPOサービスの立ち上げ責任者に従事。プレイングマネージャーとして自ら現場を牽引しながら、運用の仕組み化を行い、短期間で事業を軌道に乗せることに成功。
* **事業急拡大に伴う全方位的な採用・組織構築**：
マネジメント層から正社員、アルバイトの採用に加え、BPO領域の売上拡大に合わせた業務委託メンバーの採用を戦略的に実行。事業フェーズとコスト構造に適した人員構成を構築し、組織のスケールを支える。


#### 株式会社タイソンズアンドカンパニー (2020.03 - 2023.11)

**人事部 アシスタントマネージャー**

* **実績：** 年間100名超の採用活動を牽引。同時に、アナログな人事運用をDXツール（リプレイス含む）により「構造化」し、評価制度のオンライン化や業務効率化を実現。
* **専門性：** 急成長・多拠点組織における「制度と運用の接続」の設計。

#### 株式会社プラチナム (2019.09 - 2019.12)

**PRプランナー / コンサルタント**

* **実績：** 外資系ラグジュアリーメディアのイベントPRを担当。数値に基づく戦略設計により、昨対比248%の露出効果を創出。

#### 株式会社リクルート住まいカンパニー (2014.12 - 2019.07)

**チーフ / アドバイザー**

* **実績：** 8名のマネジメントを通じ、個人の資質に頼らない「接客ノウハウの標準化」を推進。全社の営業フローのベースとなる型を構築し、通期MVPを受賞。

## 4｜大切にしているワークスタイル

* **価値の出し方：** 「時間」の切り売りではなく、知見に基づいた「構造設計と改善」にコミットします。
* **フォーカス：** 現場100%の実行支援よりも、一段高いレイヤーでの「仕組み作り」や「アドバイザリー」に比重を置き、持続可能な組織成長を支援します。
